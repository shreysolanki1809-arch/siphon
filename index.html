Unit 4 — The High-Voltage Core

The corporation hoards every watt while the lower cities rot in the dark. You are a Siphoner. Get in, drain the grid, get out. The Warden is already awake.

Controls

  • WASD — move
  • SHIFT — run (louder, easier to spot)
  • SPACE — Holo-Belt disguise (15s, then it needs to recharge)
  • E — interact with a task console

Complete all 3 task consoles, then reach the glowing exit gate before the Warden finds you.
